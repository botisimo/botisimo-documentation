$(usernameplain)
================

Resolves the command issuer's username as plain text.

Usage:
    $(usernameplain)

Example Command:
    **name**: !hi

    **response**: Hi, $(usernameplain)!

    **twitch/youtube output**::

        user:     !hi
        botisimo: Hi, bob!

    **discord output**::

        user:     !hi
        botisimo: Hi, bob!

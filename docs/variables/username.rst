$(username)
===========

Resolves the command issuer's username as a mention.

Usage:
    $(username)

Example Command:
    **name**: !hi

    **response**: Hi, $(username)!

    **twitch/youtube output**::

        user:     !hi
        botisimo: Hi, @bob!

    **discord output**::

        user:     !hi
        botisimo: Hi, <@112338494039>!

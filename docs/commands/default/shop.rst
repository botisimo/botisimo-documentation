!shop
=====

**Default Permission:** Everyone

Display available shop items for channel in chat.

Usage:
    !shop

Example:
    ::

        user:     !shop
        botisimo: User's Shop - 1. Pizza (100,000 Points) "!buy 1" 2. Featured Account (250 Points) "!buy 2" - more at https://botisimo.com/u/user/shop

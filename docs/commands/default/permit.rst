!permit
=======

**Default Permission:** Moderator

Permit a user to bypass the whitelist domain spam filter for 1 minute.

Usage:
    !permit ``<username>``

Arguments:
    * ``username`` **<required>** - Name of the user to permit

Example:
    ::

        user:     !permit @bob
        botisimo: @bob has been permitted to send links for 1 minute

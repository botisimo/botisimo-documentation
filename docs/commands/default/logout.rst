!logout (Discord only)
======================

**Default Permission:** Everyone

Disable chat mirroring for your Discord user (platform can be ``twitch`` or ``mixer``).

Usage:
    !logout ``<platform>``

Arguments:
    * ``platform`` **<required>** - The platform to deauthorize (valid values: ``twitch``, ``mixer``)

!wikipedia
==========

**Default Permission:** Everyone

Search Wikipedia topics. Returns top 10 results in Discord and top result in Twitch & Mixer.

Usage:
    !wikipedia ``<search>``

Arguments:
    * ``search`` **<required>** - The search term(s) to use
